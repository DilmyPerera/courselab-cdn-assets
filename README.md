# CourseLab CDN Assets

This repository stores public static media assets for the CourseLab LMS project.

The assets in this repository are served through jsDelivr CDN and used by the CourseLab backend and frontend applications.

## Used For

- Course thumbnails
- Instructor profile pictures
- Default course placeholder images
- Default instructor avatar images
- Other public LMS media assets
